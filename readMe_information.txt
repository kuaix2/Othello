Ce dossier contient une implémentation en Python du jeu Othello, développée en équipe au collège, mais sans interface graphique. Voici une description des fichiers inclus :

- generalJoueur.py : Ce fichier contient les vérifications des règles du jeu pour s'assurer de leur respect.
- Joueur 1.py : Ce fichier correspond au code du joueur humain.
- joueur 2 : Ce fichier contient le code de l'IA, représentant l'ordinateur.
- Othello .py: C'est le fichier principal où le programme est exécuté.

-----------------------------------------------------

This folder contains a Python implementation of the Othello game, developed as a team in college, but without a graphical interface. Here is a description of the included files:

- generalJoueur.py: This file contains the checks of the rules of the game to ensure that they are respected.
- Joueur 1.py: This file corresponds to the code of the human player.
- joueur 2: This file contains the code of the AI, representing the computer.
- Othello .py: This is the main file where the program is executed.
